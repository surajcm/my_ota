A basic working OTA webservices connecting to IATA NDC sandbox and publishing both REST and GRAPHQL.

##### Initial Setup
Set the environment variables: 
M2_HOME, JAVA_HOME etc

##### Build command :
`./mvnw clean build`

##### To Run the application :
`./mvnw spring-boot:run`
